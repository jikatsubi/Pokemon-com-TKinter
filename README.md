Esse projetinho é um jogo simples de Pokémon feito em Python +
Tkinter. A ideia é ter uma interface gráfica onde você escolhe seu
Pokémon, luta contra outro e vê o resultado na tela.

Como rodar o projeto

1.  Certifique-se de que você tem Python 3 instalado.
2.  Baixe os arquivos do projeto e extraia tudo numa pasta.
3.  Dentro da pasta, rode o arquivo principal: python gui_game.py
4.  A janela do jogo vai abrir na tela.

Estrutura do Projeto

gui_game.py - Arquivo principal da interface gráfica pokemon.py - Classe
Pokémon e lógica do jogo

Explicando os Arquivos

pokemon.py: - Classe Pokemon (nome, tipo, vida, ataque) - Métodos de
ataque, cálculo de dano, verificação de derrota É onde ficam as regras e
lógica do jogo.

gui_game.py: - Criação da interface Tkinter - Botões, labels e imagens -
Escolha dos Pokémons - Botão de atacar - Atualização das barras de vida
e mensagens É a parte visual do jogo.

Como jogar

1.  Execute gui_game.py
2.  Escolha seu Pokémon
3.  Clique em Atacar

Requisitos

-   Python 3.x
-   Tkinter (já incluso na maioria das instalações)
